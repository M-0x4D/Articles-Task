## Deployment

To deploy this project run

1- 
```bash
  git clone https://github.com/M-0x4D/Articles-Task.git
```
2- if you are windows run this command

```bash
  copy .env.examle .env
```
- if you are linux run this command

```bash
  cp .env.examle .env
```

- and then 

```bash
  DB_DATABASE=YOUR_DATABASE_NAME

```

3-

```bash
  composer install
```

4- 

```bash
  php artisan migrate
```

4 - finally

```bash
  php artisan serve
```
